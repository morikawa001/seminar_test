# Agents

## Project Overview
RPGアドベンチャー - ブラウザで動作するドラゴンクエスト風RPG

## Tech Stack
- HTML5 / CSS3 / JavaScript (ES6+)
- Canvas API (2D描画)
- Web Audio API (サウンド生成)

## File Structure
- `index.html` - 全コード（HTML/CSS/JS統合）
- `skills.md` - プロジェクト技術書類

## Commands
- ブラウザで開く: `start index.html`
- git push: `git add . && git commit -m "message" && git push`

## Conventions
- 単一ファイル構成（外部依存なし）
- 日本語コメント・UI
- セミコロンあり

## Game Features
- タイルベースのフィールド移動
- コマンド選択式のターン制戦闘
- 5種類のモンスター + 大魔王ボス
- 魔法システム（ホイミ・メラ・ベギラマ・ビッグバン）
- アイテムシステム（ポーション）
- レベルアップ・ステータス成長
- 効果音 (Web Audio API)
- タイトル画面・ゲームオーバー画面
- セーブ機能（localStorage）
